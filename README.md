# Excecute In Development

1. Clone the Repository
2. Execute

```
yarn install
```
3. Install Nest CLI

```
npm i -g @nestjs/cli
```

4. Up The Database

```
docker-compose up -d
```
5. Recreate Database with seed
```
localhost:3000/api/v2/seed
```

## Stack

* MongoDB
* Nest
